# rps-cleanup-workshop-2024

## Setup

Create and activate a new virtual environment:

```sh
conda create -n rps-env python=3.11
conda activate rps-env
```

## Usage

Run the RPS game:

```sh
# python app/game.py

python -m app.game
```

Run the web application interface:

```sh
python -m app.web_game
```
